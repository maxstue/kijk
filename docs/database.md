# Database

For the database [neonDb](https://neon.tech/) is used with prisma as the orm.
To work with a serverless database that is also capable to branch and merge new changes, just like **git**, can be difficult but there is a good documentation on the planetscale [website](https://planetscale.com/docs/prisma/automatic-prisma-migrations).
